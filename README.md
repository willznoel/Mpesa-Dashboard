# 📊 M-Pesa Personal Finance Dashboard

## Project Overview
This project analyzes personal mobile money transactions from **M-Pesa**, one of the most widely used mobile payment platforms in Africa.  

The goal of the project was to transform raw transaction history into meaningful financial insights by cleaning, categorizing, and visualizing spending patterns.

Using Power BI, the data was transformed into an interactive dashboard that highlights income, spending behavior, and transfer activity over time.

---

## Dataset
The dataset contains personal transaction history exported from the **M-PESA Africa App**.  
Since the raw dataset contained unstructured transaction descriptions, additional preprocessing and categorization were required to make the data suitable for analysis.

---

## Tools Used
- Excel – data preparation and initial cleaning  
- Power BI – dashboard development and visualization  
- DAX – calculated measures and financial metrics

---

## What I Worked On
- Cleaned and structured raw transaction data  
- Categorized unstructured transaction descriptions into meaningful spending categories  
- Built **DAX measures** to calculate:
  - Total income
  - Total spending
  - Transfers
  - Net balance
- Designed an interactive Power BI dashboard

---

## Dashboard Visualizations
The dashboard includes several visual elements to help understand financial behavior:

- **Doughnut chart** – spending distribution by category  
- **Line chart** – transaction trends over time  
- **Bar chart** – comparison of income vs expenses  
- **KPI cards** – key financial metrics such as total income, total spending, and net balance

---

## Key Insights
The dashboard helps answer important personal finance questions such as:

- Where most spending occurs  
- How income compares to expenses  
- Transaction trends over time  
- The proportion of transfers versus direct spending
